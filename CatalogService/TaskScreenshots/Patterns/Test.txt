What is the best definition of a pattern?
A. Pattern - is the set of instructions that need to be followed in order to solve a problem.
B. Pattern - is a universal solution that can solve a vast majority of problems.
C. Pattern - is a solution to a specific type of problems that occurs repeatedly within a given context.

C

What is the correct description of software design patterns?
A. Software design patterns - are solutions which solve problems related to the integration of new and existing software in a business environment.
B. Software design patterns - are solutions which solve commonly occurring problems within a given context in software design, which typically show relationships and interactions between classes or objects.
C. Software design patterns - are solutions which solve commonly occurring problems in software architecture within a given context.
